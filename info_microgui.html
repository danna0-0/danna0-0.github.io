Conexión del Hardware

Primero, se debe conectar tu tarjeta Arduino al entrenador y asegurando de que esté correctamente alimentada y conectada a la computadora a través de USB.

Carga del Programa Base

Se utiliza el IDE de Arduino para cargar el programa base en la tarjeta Arduino. Este programa base se comunica con la interfaz gráfica y configura los pines necesarios para cada función (relevador, LEDs, LCD, sensor, potenciómetro y LEDs RGB).

Interacción con la Interfaz Gráfica

  • Encender el Relevador: Utiliza el apartado designado en la interfaz gráfica para enviar una señal al pin correspondiente de la tarjeta Arduino que activa el relevador.
  • Encender LEDs: Utiliza el apartado específico para controlar los LEDs conectados a ciertos pines de la tarjeta Arduino.
  • Enviar Mensajes al LCD: Escribe los mensajes que deseas mostrar en el LCD en el apartado de la interfaz gráfica y envíalos a la tarjeta Arduino para que los muestre en el LCD.
  • Gráfica de la Señal del Sensor: Utiliza el apartado para gráficar la señal del sensor conectado a la tarjeta Arduino. A medida que el sensor detecta cambios, la interfaz gráfica debe reflejar estos cambios en tiempo real.
  • Leer el Valor del Potenciómetro: Utiliza el apartado designado para leer el valor del potenciómetro conectado a la tarjeta Arduino y muestra este valor en la interfaz gráfica.
  • Control de PWM: Utiliza el apartado PWM para ajustar la intensidad de un LED RGB o controlar la velocidad de un motor conectado a un pin PWM de la tarjeta Arduino.
